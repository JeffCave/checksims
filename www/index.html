     What file formats are supported?
     

      MISS was originally designed to help with an introductory python class,
      and was shortly followed by a Javascript class.
     

     
    
      
  • Python (*.py)
    • 
      
  • Javascript (*.js)
    • 
      
  • FASTA (*.dna)
    • 
      
  • Plain English (*.txt)
    • 
     

     

      Files are tokenized based on their file type (identified by extension)
      which gives them some contextual interpretation.

     How do you protect my data?
     

      Very simply: I don't store any.
     

     

      One of the primary impetus for the project was to address concerns of
      retaliation from students.
      An instructor expressed concerns about transmitting data across
      international boundaries, and having it stored in data-centers not
      in the institutions control. To overcome these objectives, MISS was
      created to work entirely from within the local browser.
     

     

      No data is transferred off you computer.
     

     

      Having said that, I do collect anonymous usage data about your use of
      the site. It is hoped this information will give insight into what new
      features should be implemented.
     

    

    

     So this program tells me who is cheating?
     
No! Used properly it will tell you who is not cheating.

     

      This program does a blind comparison, but like any creative work,
      software will have common elements to it. Good solutions will be
      independantly discovered. This program does not catch cheaters.
     

     

      Rather, this program gives you a place to begin your search. Do you have
      120 papers to grade? You probably only need to investigate about 15%.
      Rather than having to search through hundreds of submissions, this can
      narrow your search down to things that look similar. MISS
      identifies the point where similarity becomes abnormally high for the
      group.
     

     

      In fact, this tool, in its first iteration, found someone suspected of
      cheating to have done nothing wrong.
     

     

      The student had been given a second chance to submit an assignment late
      and had submitted a program that was suspicously similar to a unique
      solution presented by a strong student. Events led to a more intensive
      inspection than was warranted. The fact that computers have no passion
      means they never analyze with bias, and instead showed that the level of
      similarity was high, but normal for the class (and was actually just
      tutoring).
